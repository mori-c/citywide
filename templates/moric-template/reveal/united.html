Global State + Custom Events

° ° °

< section data-state="something" > + Reveal.addEventListener({});

  • Applicable per slide or a series of slides
  • Assign "something" = turns into a class
    • e.g. switching background page
  • Custom data-state events can be made by adding this JS structure:
• • •

Reveal.addEventListener ( 'customevent', function () { console.log ( '"customevent" has fired' ); } );
